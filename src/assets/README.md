# DryerFox Assets

This folder contains the PNG and audio assets used to render the dryer:

- `dryer_lid_med.png` — The dryer lid; control buttons are overlaid on top of it.
- `dryer_frame_med.png` — The outer frame, with a transparent hole in the middle for the rotating page.
- `dryer_drum_med.png` — The drum that sits behind the page (visible briefly before the page loads).
- `dryer-sound.mp3` — Tumble sound played while a page is loading.
